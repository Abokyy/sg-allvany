# SG Állvány

Website for **SG Állvány**, a scaffolding rental company in Gödöllő, Hungary.
It rents out facade (*homlokzati*) and rolling (*gurulós*) scaffolding to
individuals and contractors, with delivery, assembly, and free on-site surveys.

Built with **React + Vite** and hosted on **GitHub Pages**. The price list is
editable without touching code, and the site includes a tawk.to chat widget.
Content is in Hungarian.
